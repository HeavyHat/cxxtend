cxxtend
=======

Introduction
------------
This is a packaged small framework with useful utilities to aid in particular tasks within C++. Currently it is only compatiable with C++14 or greater compiler but works in both clang and gcc. The utilities that are packaged are proof-of-concept and provided as-is. Developers are able to checkout the repo and make any necessary modifications that they need to accompish particular functionality. Issues can be raised to suggest enhancements to existing classes and raise awareness to particular issues.

Contributing
------------

PRs will be reviewed on an ad-hoc basis and new code should adhere to formatting attached and presented in this repo. These coding guidelines can be observed within this documentation.
